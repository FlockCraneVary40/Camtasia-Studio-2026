# Keyboard Shortcuts

| Action | Windows | macOS |
|---|---|---|
| Start / stop recording | F9 | Cmd+Shift+2 |
| Pause recording | F10 | Cmd+Shift+P |
| Split clip at playhead | S | S |
| Ripple delete | Shift+Del | Shift+Delete |
| Add marker | M | M |
| Zoom in timeline | Ctrl+= | Cmd+= |
| Export | Ctrl+E | Cmd+E |
